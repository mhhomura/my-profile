## Instalação

Instruções sobre como instalar e executar o projeto localmente. Por exemplo:

1. Clone o repositório:
```bash
git clone https://github.com/mhhomura/my-profile.git
```
2. Entre na pasta:
```bash
cd my-profile
```
3. Instale as dependencias:
```bash
yarn 
```
4. Rode o projeto
```bash
yarn dev
```
