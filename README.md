## Install

Instructions on how to install and run the project locally. For example:

1. Clone the repository:
```bash
git clone https://github.com/mhhomura/my-profile.git
```
2. get in the folder:
```bash
cd my-profile
```
3. Install the dependencies:
```bash
yarn 
```
4. Run the application
```bash
yarn dev
```
